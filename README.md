# BillSplitter
Python project for basic "Bill Splitting" functionality for a group of people

## Prerequisites

- Python 3.x

## Quick Start

To run tests, either install modules using pip from requirements.txt, or create a venv

Then run from root folder:
```bash
cd application
pytest
```

## Project Structure
```
├── src/ # Core modules for Persons, Bills, etc
├── examples/ # Usage examples for functionality
├── tests/ # pytest unit tests
├── pytest.toml # Pytest Config
├── pyproject.toml # Pyproject Config
├── .gitignore
└── README.md
```
